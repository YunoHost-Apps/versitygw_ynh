## Administration  
`__APP__` ne peut pas être installé sur un chemin.  
:wrench: L'interface d'administration est accessible à l'adresse <https://__DOMAIN____PATH__index.html>.  
:key: Le point de terminaison S3 est <https://__DOMAIN____PATH__>.
